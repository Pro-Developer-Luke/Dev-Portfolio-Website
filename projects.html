  • Digit recognition program

    Description

    I developed an AI-powered script using Python, utilising a large dataset and handwriting characteristics to detect handwritten numbers with high accuracy. The script employed machine learning algorithms, specifically convolutional neural networks (CNNs), to analyse the digitised images and identify the corresponding numbers. The algorithm was trained on a vast dataset of handwritten numbers, enabling it to recognise and learn the distinct characteristics of each digit, such as shape, size, and stroke patterns. Additionally, the script calculated the probability and confidence of each detection, providing a comprehensive output. This allowed for accurate analysis and verification of the results, as well as identification of potential errors or ambiguities. Once the script finished processing, it displayed the detected numbers, along with their corresponding probabilities and confidence levels, enabling users to make informed decisions.
